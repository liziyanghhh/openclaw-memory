## 日期计算逻辑（灵活筛选）

### 4月表格日期对应关系
| 表格行号 | 日期 | A列时间值 |
|---------|------|----------|
| 第2行 | 4月1日 | 46113 |
| 第3行 | 4月2日 | 46114 |
| 第4行 | 4月3日 | 46115 |
| ... | 以此类推 | 每次+1 |

### 公式：日期 → 行号
```
行号 = 2 + (目标日期 - 4月1日)
时间值 = 46113 + (目标日期 - 4月1日)
```

### 示例
- 用户说"4月5日" → 行号 = 2 + 4 = **第6行**，时间值 = 46113 + 4 = 46117
- 用户说"4月10日" → 行号 = 2 + 9 = **第11行**，时间值 = 46113 + 9 = 46122
- 用户说"4月15日" → 行号 = 2 + 14 = **第16行**，时间值 = 46113 + 14 = 46127
