## 使用规则

1. **执行前必读** — 每次执行定时任务前，先读取对应 playbook 文件
2. **不跳步骤** — 严格按照 playbook 步骤执行
3. **记录偏差** — 实际操作与 playbook 不一致时，立即更新文件
4. **完成后打勾** — 在 checklist 上标记完成状态
