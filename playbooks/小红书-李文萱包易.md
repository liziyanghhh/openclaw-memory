
## 关键规则

| 规则 | 说明 |
|------|------|
| 子账户名 | YX-包易二奢入门陪跑（只有一个横杠，不是 YX--） |
| 基础报表写入 JLrqHb | 第一步数据写入「26年4月」sheet |
| 笔记报表覆盖 KlQEFs | 第二步数据覆盖到「2026笔记投放数据」sheet |
| vSellerId 必须记录 | 跳转后从 URL 获取，用于直接访问报表 |
| 点击率÷100 | 平台显示5.37%，写入用0.0537 |
| 聚光平台用 JS 点击 | div 元素普通 ref 点击无效，需用 JS evaluate |

## 关键教训

1. **vSellerId 必须记录** — 跳转到聚光平台后立即从 URL 获取
2. **笔记报表在基础报表下级** — 左侧导航先展开再点击
3. **下载按钮在行最右边** — 找向下箭头图标，不是整行都能点
4. **2026笔记投放数据是独立子表** — 不是 KlQEFs，需单独切换

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 找不到笔记报表 | 没展开基础报表菜单 | 先点击「基础报表」展开，再找笔记报表 |
| 找不到下载按钮 | 滚动位置或行不对 | 确认在数据明细区域，找行右侧箭头图标 |
| 「2026笔记投放数据」子表不存在 | sheet 名称不一致 | 先查实际 sheet 名称和 ID |
| 粘贴数据乱码 | Excel 格式问题 | 全选 → 设置为文本格式 → 重新复制粘贴 |
