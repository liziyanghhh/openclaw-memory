## 关键规则

| 规则 | 说明 |
|------|------|
| **分日≠日报** | 日期筛选显示"昨天"≠日报视图，必须额外选日报 |
| **日报步骤绝对不能跳过** | 无论数据是否一致，每次都必须选日报 |
| **A列禁止写入** | 写入范围必须从B列开始 |
| **大阪表无U列** | 只有B→T（19列），不存在U列 |
| **点击率÷100** | 平台显示5.37%，写入用0.0537 |
| **聚光平台用JS点击** | div 元素普通 ref 点击无效，需用 JS evaluate |

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| **数据不准确/偏小** | 跳过了日报步骤 | 必须选日报视图 |
| 聚光平台选项点击无反应 | div 元素不是 button | 用 JS evaluate 点击 `.d-grid-item` |
| 写入后数据错位 | 包含了A列或写多了U列 | 确认范围是 B:T |
