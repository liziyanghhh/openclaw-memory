| 聚光平台用 JS 点击 | div 元素普通 ref 点击无效，需用 JS evaluate |

## 关键教训

1. **大阪账号必须登录合作伙伴平台跳转** — 不可直接访问 ad.xiaohongshu.com（需带 vSellerId）
2. **搜索框必须点击下拉选项** — 只输入文字不会过滤子账户
3. **聚光平台选项是 div 元素** — 不是标准 button，需用 JS evaluate 点击
4. **登录后可能弹出申请框** — 找到关闭按钮（×）或 aria-label="close" 关闭
5. **点击率要除以100** — 平台显示5.37%，写入用0.0537

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 跳转后是代理商后台 | 从错误入口进入 | 必须从 partner.xiaohongshu.com 进入并跳转子账户 |
| 看不到子账户数据 | 未选下拉选项 | 必须点击"YX--INNN大阪机场接送"选项 |
| 聚光平台选项点击无反应 | div 元素不是 button | 用 JS evaluate 点击 `.d-grid-item` |
| 登录后弹出"新建申请"框 | 系统自动弹窗 | 关闭对话框（×按钮）后继续 |
| 找不到合计行 | 日期筛选错误 | 确认日期设置为目标日期 |
| A列日期错误 | 缺少.1后缀 | Excel日期必须写成 46114.1 |
