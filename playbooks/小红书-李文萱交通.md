## 日期计算

- 今天 = X月Y日
- 目标日期 = X月(Y-1)日
- Excel日期：46113 = 4月1日，46114 = 4月2日
- 行号 = 目标日期 - 46111

**示例：** 今天4月3日 → 目标4月2日 → Excel日期46114 → 行号 = 46114 - 46111 = **第3行**

### 第五步：写入飞书

使用 lark-cli 写入。创建 .bat 文件：

```batch
@echo off
lark-cli sheets +write --spreadsheet-token DuyywOXW5iPUAbk2JWOcKq1RnSe --sheet-id nfuwzz --range "B{行号}:U{行号}" --values "[[消费,展现量,点击量,点击率小数,平均点击成本,千展费用,点赞,评论,关注,分享,互动量,互动成本,私信进线数,私信进线成本,私信开口数,私信开口成本,私信开口条数,私信留资数,私信留资成本,0]]"
```

**注意：** 点击率必须是**小数形式**（5.37% → 0.0537）

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 找不到合计行 | 日期筛选不对 | 确认日期筛选已设置为前一天 |
| 点击率为空 | 平台显示格式问题 | 检查原始值，可能是百分号需要转换 |
| 写入失败 | range 格式错误 | 确认行号正确，range 应为 "B3:U3" |
