## 日期计算

- 今天 = X月Y日
- 目标日期 = X月(Y-1)日
- Excel日期：46113 = 4月1日，46114 = 4月2日，46115 = 4月3日
- 行号 = Excel日期 - 46111

**示例：** 今天4月3日 → 目标4月2日 → Excel日期46114 → 行号 = 46114 - 46111 = **第3行**

- [ ] 打开 https://ad.xiaohongshu.com/

- [ ] 点击左侧边栏「数据」→「基础报表」

> ✅ 优先使用「昨天」快捷按钮

- [ ] 在日期筛选区域点击「昨天」按钮
- [ ] 确认日期显示为「昨天」的日期

### 第四步：选择「日报」视图

> ⚠️ **必须执行！此步骤绝对不能跳过！分日≠日报！**
>
> - 日期筛选显示"昨天"/"分日" ≠ 日报视图
> - 必须额外操作「细分模式」下拉，选择「日报」选项

**具体操作：**
- [ ] 滚动到「数据明细」区域

- [ ] 在下拉列表中点击「**日报**」
- [ ] 等待页面切换，**确认底部「日报」按钮高亮激活**

> **验证标准：** 页面底部「日报」按钮高亮 = 操作成功

## 关键规则

| 规则 | 说明 |
|------|------|
| **分日≠日报** | 日期筛选显示"昨天"≠日报视图，必须额外选日报 |
| **日报步骤绝对不能跳过** | 无论数据是否一致，每次都必须选日报 |

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| **数据不准确/偏小** | 跳过了日报步骤 | 必须选日报视图 |
| 数据全为0 | 账户余额不足 | 确认日期和账户状态 |
| 私信数据缺失 | 读了侧边栏概览数据 | 必须到「数据明细」读合计行 |
